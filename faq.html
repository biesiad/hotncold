Hot n'Cold

  • What is it all about?

    With this tiny application, you can play Hot n'Cold game using your mobile browser. You need mobile phone with modern web browser and GPS module, and some uncovered area where you could play.

    1. Create new game
    2. Set the secret spot
    3. Share game link with your friend
    4. Enjoy him wandering around town

  • Why should I use it?

    For fun!

  • Why did you create it?

    You guessed it... for fun!

  • Could you add tracking, profiles etc?

    Sure, catch me on twitter and let's think about come cool extensions.

  • I have privacy concerns.. Is it safe?

    Don't worry, the only trace you leave is shortened game link containing secret spot position. I'm not going to share it with anyone.
