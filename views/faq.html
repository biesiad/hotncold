• What is it all about?

    With this tiny application, you can play Hot n'Cold game using your mobile browser. You need mobile phone with modern web browser and GPS module, and some uncovered area where you could play.

    1. Create new game
    2. Set the secret spot
    3. Share game link with your friend
    4. Enjoy him wandering around town
